# EVALUACI-N---UNIDAD-03---DISTRIBUCIONES-DE-PROBABILIDAD-Y-PRUEBA-DE-HIP-TESIS__
Aplicación interactiva desarrollada en Python y Streamlit para la enseñanza de estadística inferencial. El sistema permite la carga de datos, análisis exploratorio visual (EDA) y la ejecución de pruebas de hipótesis (Z-Test) con retroalimentación en tiempo real de modelos de lenguaje de gran escala (LLM).
